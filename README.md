# Customer Segmentation and Churn Prediction Project

## Project Overview

This project performs customer segmentation and churn prediction using machine learning techniques.

## Pipeline

* Exploratory Data Analysis (EDA)
* Data Cleaning
* RFM Feature Engineering
* Customer Segmentation using KMeans
* Churn Prediction using Logistic Regression and Random Forest
* Model Evaluation

## Technologies

* Python
* Pandas
* Scikit-learn
* Machine Learning
* Data Analysis
